Habilidades

Algumas das linguagens e ferramentas que já usei em alguns projetos meus:

HTML

HTML é o bloco de construção mais básico da web. Define o significado e a estrutura do conteúdo da web.

CSS

CSS é uma linguagem de estilo usada para descrever a apresentação de um documento escrito em HTML. O CSS descreve como elementos são mostrados.

Javascript

JavaScript é uma linguagem de programação interpretada estruturada, de script em alto nível com tipagem dinâmica fraca e multiparadigma.

React

React é uma biblioteca front-end JavaScript de código aberto com foco em criar interfaces de usuário em páginas web.

Typescript

TypeScript é um conjunto de ferramentas e formas mais eficientes de escrever em JavaScript, adicionando recursos não presentes de maneira nativa na linguagem.

Python

Python é uma linguagem de programação conhecida por sua sintaxe simples e legível. É usada em automação, web, ciência de dados e inteligência artificial.

Java

Java é uma inguagem orientada a objetos usada no desenvolvimento de aplicativos, sistemas e plataformas web. É conhecida por sua portabilidade e robustez.

Figma

Figma é uma ferramenta poderosa para criar designs de interfaces, protótipos interativos e projetos de UI/UX, com colaboração em tempo real e recursos pra organizar ideias visuais.

Git

Git é usado para controlar o histórico de alterações de arquivos e principalmente de projetos de desenvolvimento de software.
